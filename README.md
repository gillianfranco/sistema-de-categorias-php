# Sistema de Categorias em PHP

Projeto desenvolvido com o objetivo de exercitar a implementação de uma estrutura MVC do zero, a manipulação de um banco de dados MySQL hospedado em um container e a utilização do jQuery para a execução de requisições assíncronas.

Funcionalidades implementadas:
- Cadastro, edição e exclusão de **categorias**
- Cadastro, edição e exclusão de **produtos**
- Exibição de categorias e produtos cadastrados, organizados hierarquicamente por categoria
